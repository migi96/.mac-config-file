## Commands
- **Build Commands**:
  - `make tiktoken`, `make install_jsregexp`, `npm i`
  - Auto-run setup: `auto_run_build_runner`, `:TSUpdate`, `custom build functions`
- **Linting**:
  - `eslint_d` for JS/TS-based files like `react`, `svelte`
  - `pylint` for Python via `lint.try_lint()`
- **Testing**:
  - `Flutter: <leader>ct` (new test creation).
